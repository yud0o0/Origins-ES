# Origins-ES Addon (v1.21.1)

This is a custom Fabric addon for the Origins mod, developed specifically for a private server. The source code is provided for educational purposes or for those looking to implement similar mechanics in their own projects.

## Technical Specifications
- **Game Version:** 1.21.1
- **Loader:** Fabric
- **Language:** Java 21
- **Required Dependencies:** Origins (Apoli), Fabric API

## Disclaimer
This mod was developed for private use. Code structure and naming conventions are optimized for internal server requirements rather than public distribution.
